# form-renderer 领域事件

## 发布事件

本子模块不发布领域事件。

- `form-renderer` 是纯前端渲染层，不拥有后端权威数据，也不负责跨模块状态变更。
- 表单联动、校验和视图切换仅影响浏览器内临时状态，不应进入事件总线。

## 消费事件

本子模块也不直接消费事件总线事件。

- 渲染器通过同步 HTTP 接口获取渲染协议和运行态上下文。
- 若页面需要实时刷新，统一由页面重新拉取上游接口，而不是在浏览器内订阅领域事件。

## 投影、幂等与补偿

- 无独立投影模型。
- 无领域事件幂等处理。
- 页面重试、重新拉取和附件上传补偿均依赖上游接口，不在本模块定义补偿链路。
