# 06-消息移动与生态-模块功能

## 模块定位

该模块负责统一消息通知、待办提醒、移动办公能力和第三方生态连接能力，是提升系统触达效率、移动处理效率和对外集成能力的重要支撑模块。它需要把平台内各类业务变化及时、准确地传递到用户和外部系统。

## 业务目标

- 建立统一消息中心和统一提醒机制
- 支持站内信、邮件、短信、企业微信、钉钉、APP Push 等多渠道触达
- 支持移动端待办处理、消息查看、内容阅读和文件预览
- 支持与企业微信、钉钉、SSO、LDAP、第三方业务系统等集成
- 支持消息模板、发送策略、失败重试和送达追踪

## 边界约束

- 本模块负责用户触达、消息路由、移动端入口、SSO 与生态登录接入，不负责平台级通用数据开放和连接器治理。
- 与企业微信、钉钉、SSO、LDAP 相关的登录接入、消息渠道、移动工作台入口属于本模块范围。
- 开放 API、Webhook、通用连接器、跨系统数据同步和服务治理统一由 `07-数据服务与集成` 承担。

## 子模块划分

### 6.1 统一消息中心

负责消息模型和消息状态管理。

| 功能点 | 后端能力 | 前端页面/能力 |
|--------|----------|---------------|
| 消息模型管理 | 消息类型（流程通知/审批提醒/系统通知/公告/日程/会议/任务/到期预警）、消息 CRUD | 消息列表页、消息详情页 |
| 消息状态管理 | 未读/已读/撤回/失效/归档状态流转 | 一键已读、批量处理、状态标签 |
| 消息筛选 | 按类型/时间/状态/来源模块筛选 | 筛选面板、分类 Tab |

### 6.2 多渠道发送能力

负责消息内容与渠道发送解耦。

| 功能点 | 后端能力 | 前端页面/能力 |
|--------|----------|---------------|
| 渠道路由 | 站内信/邮件/短信/企业微信/钉钉/APP Push 渠道路由 | 渠道配置页、路由规则编辑 |
| 消息模板 | 模板定义、变量替换、模板预览 | 模板管理页、模板编辑器 |
| 发送策略 | 发送优先级、发送频率控制、静默规则 | 策略配置页 |
| 失败重试与补偿 | 自动重试、补偿机制、人工干预 | 重试记录页、人工干预入口 |
| 送达追踪 | 送达状态、送达时间、送达渠道记录 | 送达状态页、追踪详情 |

### 6.3 待办提醒与事件订阅

负责业务事件接入和提醒生成。

| 功能点 | 后端能力 | 前端页面/能力 |
|--------|----------|---------------|
| 事件接入 | 流程/内容/会议/日程/合同/资产等模块事件接入 | 事件源配置页 |
| 订阅偏好 | 用户订阅偏好、静默规则、提醒频率控制 | 订阅偏好设置页 |
| 高优先级升级 | 高优先级待办更强触达方式、升级策略 | 优先级标签、升级提示 |
| 审计与告警 | 消息审计、异常告警 | 审计日志页、告警列表 |

### 6.4 移动端接入支持

负责移动端轻量 API 和安全能力。

| 功能点 | 后端能力 | 前端页面/能力 |
|--------|----------|---------------|
| 移动端聚合接口 | 轻量 API、消息拉取/推送、高频数据聚合 | 移动端首页、消息页 |
| 登录态管理 | 设备绑定、登录安全、会话续期、风险控制 | 移动端登录页、安全验证 |
| 弱网容错 | 离线缓存、请求队列、重试策略 | 离线提示、重试交互 |

### 6.5 第三方生态集成

负责与企业微信、钉钉、SSO 等系统对接。

| 功能点 | 后端能力 | 前端页面/能力 |
|--------|----------|---------------|
| 企业微信/钉钉接入 | 回调处理、签名校验、租户隔离 | 生态配置页、参数维护 |
| 短信/邮件平台接入 | 渠道参数配置、发送测试 | 配置页、发送测试按钮 |
| SSO/LDAP 接入 | 单点登录、组织目录同步 | SSO 配置页、LDAP 配置页 |
| 接入状态与日志 | 接入状态监控、异常日志、审计 | 状态监控页、异常排查入口 |

## 优化方向

- 用统一事件和消息模型支撑所有业务模块，避免重复通知逻辑
- 将消息提醒和待办中心打通，减少用户信息分散
- 优先强化移动端高频链路，而不是简单搬运 PC 页面
- 对外集成能力平台化，减少项目级重复开发
- 加强消息模板、发送策略和触达分析能力

## 典型业务场景

- 用户收到审批待办消息后直接进入移动端审批页面
- 会议开始前向参会人员推送消息提醒和签到入口
- 公告发布后向目标组织发送站内消息和企业微信通知
- 合同到期前自动触发多渠道预警
- 企业微信单点登录后直接进入个人工作台并查看未读消息

## 交付边界

该模块不直接承载业务实体本身，但承担平台事件触达、移动入口和外部生态连接的中枢角色，对整体使用率和效率影响极大。它负责的是面向用户的触达与接入能力，而不是面向第三方系统的通用数据开放能力。
