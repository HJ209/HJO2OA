# Connector 接口说明

## 接口分组

| 接口分组 | 关键查询 | 关键写入 | 分页与筛选 | 幂等要求 | 权限要求 | 审计要求 | 错误处理 |
|----------|----------|----------|------------|----------|----------|----------|----------|
| 连接器目录 | 查询连接器列表、详情、最近测试结果 | 新增连接器、更新基础信息、启用/停用 | 支持按连接器类型、状态、编码、关键字分页筛选 | 创建/状态切换需带请求幂等键，避免重复创建或重复启停 | 仅平台集成管理员可写，授权查看人可读 | 记录新增、修改、启停操作人和前后差异 | 参数校验失败、状态冲突、编码重复需返回标准错误码 |
| 参数管理 | 查询参数模板、参数详情、敏感项标记 | 保存参数、更新密钥引用、批量校验必填项 | 支持按参数类别、是否敏感过滤 | 同一连接器同一参数键写入需幂等覆盖 | 仅具备安全配置权限的管理员可写 | 审计参数键变更，不记录敏感明文 | 缺少密钥引用、必填缺失、非法协议返回可定位错误 |
| 连接测试 | 查询测试历史、最近一次耗时和结果 | 发起手工测试、确认测试通过后启用 | 支持按时间范围、结果状态筛选历史记录 | 测试请求按连接器+请求时间窗去重，避免重复压测 | 仅运维或集成管理员可执行 | 记录测试发起人、目标环境、结果摘要 | 超时、认证失败、网络不可达统一归类，不暴露敏感细节 |
| 健康快照 | 查询健康状态、最近失败原因、趋势摘要 | 刷新健康快照、人工确认异常 | 支持按状态、最近失败时间筛选 | 同一检查窗口内重复刷新仅保留最新有效结果 | 管理员可写，只读角色可看状态 | 记录人工确认与备注 | 依赖不可用、快照缺失、检查任务未完成需明确区分 |

## 接口治理约束

| 项 | 约束说明 |
|----|----------|
| 查询口径 | 列表接口统一返回分页信息、筛选条件回显、最近一次测试摘要 |
| 写入口径 | 写接口只维护连接器定义与参数引用，不直接写入业务系统数据 |
| 幂等策略 | 创建、启停、测试触发等写操作需支持请求幂等键或业务唯一键去重 |
| 权限模型 | 采用平台管理权限，不向普通业务用户开放写权限 |
| 审计留痕 | 基础信息、参数引用、状态变更、连接测试均需写入统一审计 |
| 错误模型 | 使用统一错误码、错误级别和可读提示，不回传密码、Token、连接串明文 |
