# Report 前端说明

## 页面清单

| 页面/区域 | 页面职责 | 权限要求 | 状态要点 | 移动端约束 |
|-----------|----------|----------|----------|------------|
| 报表定义列表页 | 查看报表目录、类型、刷新状态、可见范围 | 报表管理员、审核人 | 草稿、启用、归档、快照过期 | 移动端仅建议只读查看 |
| 指标与维度配置页 | 维护指标、维度、可见范围和刷新方式 | 报表管理员 | 配置中、校验失败、可发布 | 移动端不提供完整编辑体验 |
| 分析结果页 | 查看趋势、排行、汇总结果并切换维度 | 报表查看人、报表管理员 | 数据新鲜、数据过期、刷新中 | 移动端保留摘要查看，不支持复杂筛选 |
| 图卡输出与快照页 | 查看图卡数据源、快照历史、新鲜度和手工刷新 | 报表管理员、运维管理员 | 可消费、刷新中、失败、过期 | 移动端仅支持查看快照状态 |

## 页面职责与关键交互

| 页面/区域 | 关键交互 | 说明 |
|-----------|----------|------|
| 报表定义列表页 | 条件筛选、查看最近刷新、跳转配置 | 列表需突出类型、来源范围、新鲜度 |
| 指标与维度配置页 | 配置指标、维度、口径说明、可见范围 | 配置变更需提示影响已有快照和图卡 |
| 分析结果页 | 时间切换、组织维度切换、趋势与排行联动 | 只展示有权限的数据范围，不暴露越权明细 |
| 图卡输出与快照页 | 查看卡片摘要、手工刷新、查看历史快照 | 页面只管理数据协议，不包含门户布局设置 |

## 前端约束

| 项 | 说明 |
|----|------|
| 权限 | 必须区分报表配置、报表刷新、结果查看权限 |
| 状态 | 前端需明确展示快照新鲜度、刷新状态和报表启停状态 |
| 移动端 | 一期移动端只保留分析摘要和快照状态查看，不支持指标配置与复杂钻取 |
| 边界 | 前端只服务统计配置和管理分析，不承担门户页面聚合和用户消息触达 |
