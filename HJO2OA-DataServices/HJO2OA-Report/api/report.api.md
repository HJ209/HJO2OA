# Report 接口说明

## 接口分组

| 接口分组 | 关键查询 | 关键写入 | 分页与筛选 | 幂等要求 | 权限要求 | 审计要求 | 错误处理 |
|----------|----------|----------|------------|----------|----------|----------|----------|
| 报表定义管理 | 查询报表列表、详情、指标和维度结构 | 新增报表、更新定义、启停报表 | 支持按报表类型、来源范围、状态、可见范围分页筛选 | 创建和启停按报表编码幂等 | 报表管理员可写，审核人可读 | 审计定义变更、可见范围和刷新策略变化 | 编码冲突、指标缺失、可见范围非法需标准返回 |
| 指标与维度配置 | 查询指标清单、维度清单、公式说明 | 维护指标、调整聚合方式、配置维度 | 支持按指标类型、维度类型筛选 | 同一报表同一指标编码写入幂等 | 报表管理员可写 | 审计公式、聚合方式和维度口径变化 | 非法公式、重复指标、维度冲突需明确提示 |
| 查询与快照 | 查询分析结果、趋势排行、快照历史、新鲜度状态 | 触发刷新、确认快照有效性 | 支持按时间范围、组织维度、状态分页筛选 | 同一报表同一刷新批次幂等 | 报表管理员、运维管理员可执行；授权查看人可读 | 审计刷新动作、快照状态变化 | 数据源不可用、快照过期、权限不足需区分 |
| 图卡数据源 | 查询图卡摘要、卡片配置关联、刷新时间 | 手工刷新图卡快照、更新卡片输出映射 | 支持按卡片类型、报表编码筛选 | 同一报表同一卡片输出刷新幂等 | 报表管理员可写，门户消费方只读 | 审计图卡输出配置和手工刷新 | 只返回卡片协议错误，不暴露门户内部实现细节 |

## 接口治理约束

| 项 | 约束说明 |
|----|----------|
| 查询口径 | 列表接口统一返回报表类型、刷新时间、新鲜度和可见范围摘要 |
| 写入口径 | 写接口只管理统计口径与快照，不直接承担门户页面配置或业务数据编辑 |
| 幂等策略 | 刷新、启停、快照确认必须绑定报表编号和刷新批次实现幂等 |
| 权限模型 | 至少区分报表配置、报表刷新、结果只读三类权限 |
| 审计留痕 | 报表定义、指标公式、可见范围、刷新动作、图卡输出配置都必须审计 |
| 错误模型 | 区分数据源错误、权限错误、口径错误、快照过期、刷新冲突，不把快照错误误判为业务源数据错误 |
