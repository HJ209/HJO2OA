# DataSync 接口说明

## 接口分组

| 接口分组 | 关键查询 | 关键写入 | 分页与筛选 | 幂等要求 | 权限要求 | 审计要求 | 错误处理 |
|----------|----------|----------|------------|----------|----------|----------|----------|
| 同步任务管理 | 查询任务列表、详情、启停状态、依赖连接器 | 新增任务、更新任务、启停任务 | 支持按同步模式、状态、源端、目标端、编码分页筛选 | 创建与启停按任务编码+请求键幂等 | 集成管理员可写，运维和审核人可读 | 审计任务定义、状态和调度变更 | 状态冲突、依赖缺失、编码重复需标准返回 |
| 映射与检查点 | 查询字段映射、检查点模式、最近检查点 | 更新映射规则、设置冲突策略、重置检查点 | 支持按字段、冲突策略、更新时间筛选 | 同一任务同一字段映射写入需幂等覆盖 | 集成管理员可写 | 审计映射和检查点修改记录 | 映射冲突、检查点非法回退需明确提示 |
| 执行与重试 | 查询执行记录、失败原因、耗时、差异摘要 | 手工执行、失败重试、暂停当前批次 | 支持按时间、执行状态、触发方式分页筛选 | 执行请求按任务+触发批次幂等，避免重复运行 | 运维管理员、集成管理员可执行 | 记录触发人、执行批次、输入上下文 | 重复执行、依赖不可用、目标写入失败需区分 |
| 对账与补偿 | 查询差异明细、补偿状态、人工处理结果 | 发起对账、提交人工补偿结论、确认忽略差异 | 支持按差异类型、处理状态、任务编码分页筛选 | 同一差异单只允许一次有效补偿结论 | 运维管理员、审核人可处理 | 审计补偿动作、处理意见和责任人 | 不允许无审计的补偿覆盖，失败需保留原始差异 |

## 接口治理约束

| 项 | 约束说明 |
|----|----------|
| 查询口径 | 列表统一返回执行摘要、最近检查点、最近失败原因和差异计数 |
| 写入口径 | 写接口只管理同步任务及其运行控制，不直接暴露业务主数据批量编辑能力 |
| 幂等策略 | 执行、重试、补偿必须绑定任务编号、执行批次或差异编号实现幂等 |
| 权限模型 | 至少区分配置权限、执行权限、审核权限，避免同人绕过审计完成定义与补偿 |
| 审计留痕 | 定义修改、手工执行、失败重试、检查点重置、补偿确认都必须审计 |
| 错误模型 | 区分源端错误、目标端错误、映射错误、检查点错误、权限错误，不隐藏失败阶段 |
