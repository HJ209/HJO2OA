# DataSync 前端说明

## 页面清单

| 页面/区域 | 页面职责 | 权限要求 | 状态要点 | 移动端约束 |
|-----------|----------|----------|----------|------------|
| 同步任务列表页 | 查看任务目录、状态、最近执行结果和差异摘要 | 集成管理员、运维管理员、审核人 | 草稿、运行中、暂停、异常 | 移动端仅建议查看状态和最近结果 |
| 同步任务配置页 | 配置源端/目标端、模式、调度、检查点和映射规则 | 集成管理员 | 草稿保存、校验失败、可执行 | 表单复杂，移动端不支持完整编辑 |
| 执行记录与对账页 | 查看执行批次、检查点、差异、失败原因 | 运维管理员、审核人 | 成功、失败、补偿中、已忽略 | 移动端只读查看，不支持复杂筛选操作 |
| 补偿处理页 | 手工重试、差异确认、提交补偿结论 | 运维管理员、审核人 | 待处理、处理中、已完成、已驳回 | 移动端不开放补偿处理，只显示处理结果 |

## 页面职责与关键交互

| 页面/区域 | 关键交互 | 说明 |
|-----------|----------|------|
| 同步任务列表页 | 条件筛选、启停切换、跳转最近失败批次 | 列表需突出同步模式、检查点和最近失败时间 |
| 同步任务配置页 | 选择连接器、维护映射、设置冲突策略 | 配置保存前必须完成依赖校验和映射完整性检查 |
| 执行记录与对账页 | 查看批次明细、比较差异、按错误分类过滤 | 差异列表需支持按字段、记录键、处理状态过滤 |
| 补偿处理页 | 一键重试、提交人工结论、附带备注 | 每次补偿都要展示对应审计链路与责任人 |

## 前端约束

| 项 | 说明 |
|----|------|
| 权限 | 必须区分任务配置、任务执行、补偿审核三类权限 |
| 状态 | 页面需同时显示任务定义状态与执行批次状态，避免概念混淆 |
| 移动端 | 一期仅保留运行态查看，不支持任务配置、映射编排、补偿提交 |
| 边界 | 前端只服务平台集成运营，不承担用户门户、开发者市场或业务主数据维护入口 |
