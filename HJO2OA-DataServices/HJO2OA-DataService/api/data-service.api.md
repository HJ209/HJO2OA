# DataService 接口说明

## 接口分组

| 接口分组 | 关键查询 | 关键写入 | 分页与筛选 | 幂等要求 | 权限要求 | 审计要求 | 错误处理 |
|----------|----------|----------|------------|----------|----------|----------|----------|
| 服务定义管理 | 查询服务列表、详情、引用关系 | 新增服务、更新服务、启用/停用服务 | 支持按服务类型、状态、编码、关键字分页筛选 | 创建和状态切换需按服务编码+请求键幂等 | 平台数据服务管理员可写，授权查看人可读 | 审计定义变更、状态变更、引用影响 | 编码重复、状态冲突、非法类型需标准返回 |
| 参数模型管理 | 查询参数清单、校验规则、分页参数说明 | 新增/更新参数、设置默认值、启停参数 | 支持按必填、参数类型、分页类参数过滤 | 同一服务同一参数编码写入需幂等覆盖 | 数据服务管理员可写 | 审计参数结构变更与默认值变化 | 必填缺失、类型不兼容、分页参数冲突需明确提示 |
| 字段映射管理 | 查询映射清单、脱敏规则、输出结构预览 | 维护映射、更新转换规则、标记脱敏 | 支持按来源字段、目标字段、脱敏状态过滤 | 同一来源字段到目标字段写入需幂等 | 数据服务管理员可写 | 审计字段映射前后差异 | 映射冲突、目标字段重复、转换规则非法需可定位 |
| 服务测试与预览 | 查询测试结果、预览输出结构 | 发起测试、保存测试基线 | 支持按时间、执行结果筛选测试历史 | 同一服务同一测试批次去重，避免重复执行 | 数据服务管理员、运维管理员可执行 | 记录测试人、输入参数、结果摘要 | 上游依赖不可用、参数校验失败、权限不足需分类返回 |

## 接口治理约束

| 项 | 约束说明 |
|----|----------|
| 查询口径 | 列表类接口统一支持分页、筛选条件回显、最近启用状态与引用数量摘要 |
| 写入口径 | 写接口只维护服务协议及其子对象，不直接暴露面向第三方的鉴权与配额能力 |
| 幂等策略 | 创建、启停、测试等写操作需支持请求幂等键，避免重复发布或重复测试 |
| 权限模型 | 权限至少区分平台管理员、服务配置管理员、只读审核人 |
| 审计留痕 | 服务定义、参数、字段映射、权限模式、缓存策略和测试动作都必须记录 |
| 错误模型 | 使用统一错误码，区分参数错误、权限错误、依赖错误、状态错误，不回传敏感底层实现细节 |
