# OpenApi 接口说明

## 接口分组

| 接口分组 | 关键查询 | 关键写入 | 分页与筛选 | 幂等要求 | 权限要求 | 审计要求 | 错误处理 |
|----------|----------|----------|------------|----------|----------|----------|----------|
| 接口目录管理 | 查询接口列表、详情、版本关系、引用服务定义 | 注册接口、更新接口、发布/废弃接口 | 支持按路径、方法、版本、状态、服务编码分页筛选 | 注册和发布按接口编码+版本幂等 | 开放接口管理员可写，审核人可读 | 审计发布、废弃、路径和授权范围变更 | 编码/路径冲突、版本冲突、依赖服务缺失需标准返回 |
| 凭证与授权 | 查询调用方凭证、授权范围、有效期 | 新增凭证、吊销凭证、调整接口授权 | 支持按调用方、状态、到期时间分页筛选 | 同一调用方同一接口授权写入需幂等覆盖 | 安全管理员、开放接口管理员可写 | 审计凭证发放、吊销和授权变更 | 凭证过期、授权缺失、签名方式不支持需明确提示 |
| 配额与限流 | 查询配额规则、限流命中统计、接口阈值 | 设置配额、调整限流、启停规则 | 支持按接口、调用方、规则状态筛选 | 同一接口同一规则编码写入幂等 | 开放接口管理员可写 | 审计阈值调整和规则启停 | 超限、规则冲突、阈值非法需标准化反馈 |
| 调用审计 | 查询调用日志、错误分布、耗时摘要 | 标记异常调用、人工备注、补录审核结论 | 支持按接口、调用方、状态码、时间范围分页筛选 | 审计补录按调用记录编号幂等 | 运维管理员、审核人可处理 | 所有审计修订和备注需留痕 | 仅允许修订审计结论，不允许篡改原始调用结果 |

## 接口治理约束

| 项 | 约束说明 |
|----|----------|
| 查询口径 | 列表接口统一返回版本、鉴权方式、调用状态摘要和最近告警信息 |
| 写入口径 | 写接口只管理开放边界、凭证和控制策略，不直接承担业务数据维护 |
| 幂等策略 | 发布、废弃、凭证发放、配额调整必须支持业务键幂等 |
| 权限模型 | 至少区分开放配置权限、安全凭证权限、审计查看权限 |
| 审计留痕 | 接口发布、废弃、授权、配额、限流、人工审计修订必须全量审计 |
| 错误模型 | 统一错误码，区分鉴权失败、授权不足、限流、配额超限、依赖服务不可用、版本不存在 |
