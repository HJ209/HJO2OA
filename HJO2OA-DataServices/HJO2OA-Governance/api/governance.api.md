# Governance 接口说明

## 接口分组

| 接口分组 | 关键查询 | 关键写入 | 分页与筛选 | 幂等要求 | 权限要求 | 审计要求 | 错误处理 |
|----------|----------|----------|------------|----------|----------|----------|----------|
| 版本治理 | 查询版本列表、发布状态、兼容关系 | 登记版本、发布版本、废弃版本 | 支持按对象类型、状态、版本号、发布时间分页筛选 | 发布/废弃按目标对象+版本号幂等 | 治理管理员可写，审核人可读 | 记录发布人、审批信息、兼容说明 | 版本冲突、重复发布、非法废弃需标准返回 |
| 健康检查规则 | 查询健康规则、目标对象、最近执行结果 | 新增规则、调整阈值、启停规则 | 支持按对象类型、状态、规则级别筛选 | 同一对象同一规则编码写入幂等 | 治理管理员可写 | 审计阈值、状态与范围变化 | 规则冲突、阈值非法、目标不存在需明确提示 |
| 告警中心 | 查询告警列表、告警详情、处理链路 | 确认告警、升级告警、关闭告警 | 支持按等级、状态、目标编码、时间范围分页筛选 | 同一告警记录的处理动作按告警编号幂等 | 运维管理员、治理管理员可处理 | 记录确认人、升级人、关闭原因 | 不允许无状态迁移规则的直接关闭 |
| 干预动作与追踪 | 查询异常链路、关联批次、人工处理历史 | 触发停用建议、重试建议、备注结论 | 支持按对象类型、异常类型、处理状态筛选 | 同一异常链路的同类干预动作幂等 | 运维管理员可执行，审核人可查看 | 全量记录动作原因、目标对象、执行结果 | 依赖状态未满足、目标已变更时需阻断并提示 |

## 接口治理约束

| 项 | 约束说明 |
|----|----------|
| 查询口径 | 所有列表接口统一返回对象标识、当前状态、最近异常、处理摘要 |
| 写入口径 | 只允许对治理规则和治理动作进行写入，不允许绕过子模块接口直接改底层定义 |
| 幂等策略 | 发布、告警确认、人工干预必须按对象版本或告警编号幂等 |
| 权限模型 | 至少区分治理配置、告警处理、只读审计三类权限 |
| 审计留痕 | 所有规则变更、状态切换和人工干预必须全量审计 |
| 错误模型 | 统一区分规则错误、状态错误、依赖错误、权限错误和审计缺失错误 |
