# Governance 前端说明

## 页面清单

| 页面/区域 | 页面职责 | 权限要求 | 状态要点 | 移动端约束 |
|-----------|----------|----------|----------|------------|
| 版本治理页 | 查看对象版本、发布状态、兼容说明和废弃计划 | 治理管理员、审核人 | 草稿、已发布、已废弃、待下线 | 移动端仅支持查看版本状态 |
| 健康监控页 | 查看 API、连接器、同步、报表的健康状态与异常摘要 | 运维管理员、治理管理员 | 正常、告警、降级、停用建议 | 移动端仅展示概览卡片和异常摘要 |
| 告警中心页 | 查看告警列表、等级、处理链路并进行确认 | 运维管理员、治理管理员、审核人 | 待确认、处理中、已关闭、已升级 | 移动端只允许确认查看，不允许复杂处理 |
| 异常追踪与干预页 | 查看关联批次、错误分类、人工备注和干预动作 | 运维管理员 | 待处理、处理中、处理完成 | 移动端不开放干预动作执行 |

## 页面职责与关键交互

| 页面/区域 | 关键交互 | 说明 |
|-----------|----------|------|
| 版本治理页 | 发布、废弃、查看兼容说明 | 发布前需显示影响对象和审计确认 |
| 健康监控页 | 筛选对象类型、查看最近异常、跳转明细 | 概览页只展示治理摘要，不承载底层配置编辑 |
| 告警中心页 | 确认告警、升级处理、关闭告警 | 每个动作都必须关联原因与责任人 |
| 异常追踪与干预页 | 查看链路、填写备注、发起受控干预 | 干预动作必须展示审计提示与不可逆风险说明 |

## 前端约束

| 项 | 说明 |
|----|------|
| 权限 | 必须区分治理配置、告警处理、只读审计和受控干预权限 |
| 状态 | 统一展示健康等级和处理状态，避免与来源模块业务状态混淆 |
| 移动端 | 一期移动端只看概览、告警和状态，不支持规则编辑、发布、干预 |
| 边界 | 治理前端只服务 `07` 开放能力运行管理，不建设统一消息中心或业务监控总台 |
