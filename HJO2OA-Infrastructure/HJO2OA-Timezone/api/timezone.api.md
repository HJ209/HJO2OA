# Timezone 接口说明

## 接口分组

| 接口分组 | 关键查询 | 关键写入 | 分页与筛选 | 幂等要求 | 权限要求 | 审计要求 | 错误处理 |
|----------|----------|----------|------------|----------|----------|----------|----------|
| 作用域时区策略管理 | 查询系统、租户、用户级时区策略、优先级链和生效窗口 | 新增/修改作用域时区、启停策略、设置默认和未来生效 | 支持按 `scopeType`、`scopeId`、`timezoneId`、`active`、`effectiveFrom` 分页筛选 | 按 `scopeType + scopeId + effectiveFrom` 幂等写入 | 平台管理员、租户管理员可写 | 默认值切换、未来生效、启停都必须审计 | `409` 生效窗口冲突，`422` 时区标识非法，`404` 作用域不存在 |
| 用户偏好管理 | 查询当前用户有效时区、作用域回退链和历史变更 | 用户自助设置时区、管理员代设、清理个人覆盖 | 支持按 `personId`、`tenantId`、`timezoneId`、`status` 筛选 | 按 `personId + timezoneId + effectiveAt` 幂等；重复设置返回当前生效结果 | 用户可改本人；管理员可代设租户内用户 | 管理员代设必须审计；个人修改至少保留摘要审计 | `403` 越权代设，`422` 时区未启用，`404` 用户不存在 |
| 转换与校验 | 查询支持的 IANA 时区、DST 规则、转换预览、批量转换结果 | 发起批量转换预览、校验导入时间、刷新转换缓存 | 支持按 `timezoneId`、`dateRange`、`dstFlag`、`sceneCode` 筛选 | 批量预览按请求摘要幂等 | 已登录用户可查预览；批量校验仅管理员或系统账号可用 | 批量校验和人工刷新必须记录触发来源 | `400` 时间格式非法，`422` DST 歧义未指定策略，`404` 时区不存在 |
| 生效窗口与治理 | 查询时区变更历史、未来生效任务、异常冲突记录 | 撤销未来生效、确认冲突处理、补录治理备注 | 支持按 `scopeType`、`scopeId`、`effectiveStatus`、时间区间筛选 | 撤销和确认操作按 `settingId + operationNo` 幂等 | 平台管理员、审计员可写 | 所有撤销、确认和备注必须进入统一审计 | `409` 设置已生效不可撤销，`404` 记录不存在，`422` 冲突未解决 |

## 接口治理约束

| 项 | 约束说明 |
|----|----------|
| 查询口径 | 统一返回当前有效时区、来源层级、未来生效记录和最近修改时间，避免前端自行拼装优先级 |
| 写入口径 | 写接口只管理时区策略和偏好，不直接修改业务表中的时间字段 |
| 幂等策略 | 默认时区切换、用户偏好写入、未来生效配置和撤销都必须支持业务键幂等 |
| 权限模型 | 至少区分平台级时区治理、租户级时区治理、个人偏好修改三类权限 |
| 审计留痕 | 系统级和租户级时区调整、未来生效、撤销冲突处理必须进入统一审计 |
| 错误模型 | 统一错误码，区分时区标识非法、作用域缺失、生效窗口冲突、DST 歧义、权限不足五类错误 |
