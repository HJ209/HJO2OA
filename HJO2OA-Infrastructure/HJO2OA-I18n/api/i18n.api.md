# I18n 接口说明

## 接口分组

| 接口分组 | 关键查询 | 关键写入 | 分页与筛选 | 幂等要求 | 权限要求 | 审计要求 | 错误处理 |
|----------|----------|----------|------------|----------|----------|----------|----------|
| 语言目录与回退策略 | 查询启用语言、默认语言、回退链、租户可用语言清单 | 新增语言、启停语言、设置默认语言、调整回退链 | 支持按 `locale`、`status`、`scopeType`、`isDefault` 分页筛选 | 按 `locale` 幂等建档；默认语言切换按 `scopeType + scopeId + effectiveAt` 幂等 | 平台管理员、租户管理员可写 | 默认语言、回退链、启停变更必须审计 | `409` 语言重复，`422` 回退链成环，`404` 语言不存在 |
| 资源包与资源项管理 | 查询资源包列表、版本详情、键值历史、差异对比 | 新建资源包、编辑资源项、发布/废弃资源包、回滚版本 | 支持按 `bundleCode`、`moduleCode`、`locale`、`status`、`keyPrefix`、更新时间筛选 | 按 `bundleCode + locale` 幂等建档；资源项按 `bundleId + resourceKey` 幂等更新 | I18n 管理员可写，审计员只读 | 批量导入、发布、废弃、回滚必须记录批次号与原因 | `409` 资源键冲突，`422` 占位符非法或格式错误，`404` 资源包不存在 |
| 导入导出与校验 | 查询导入批次、校验结果、缺失键和未翻译摘要 | 批量导入、批量导出、执行占位符校验、触发缺失键扫描 | 支持按 `batchNo`、`moduleCode`、`locale`、`validationStatus`、时间区间筛选 | 导入按 `batchNo` 或 `fileChecksum` 幂等；重复导出按请求摘要幂等 | I18n 管理员、审计员可用 | 导入、导出、校验结果确认都必须留痕 | `400` 文件格式非法，`422` 键重复或语言不支持，`409` 批次重复处理中 |
| 语言偏好与解析 | 查询个人/租户当前语言、解析预览、缺失键预览 | 设置个人语言、管理员代设作用域语言、清理个人覆盖 | 支持按 `scopeType`、`scopeId`、`locale`、`moduleCode` 筛选 | 偏好写入按 `scopeType + scopeId + locale + effectiveAt` 幂等 | 用户可改本人偏好；管理员可改租户或系统级偏好 | 系统级、租户级偏好调整必须审计；个人变更至少记录摘要 | `404` 资源键不存在，`422` 语言未启用或回退不可达 |

## 接口治理约束

| 项 | 约束说明 |
|----|----------|
| 查询口径 | 列表接口统一返回语言状态、回退目标、资源包版本和最近更新时间，避免前端二次拼装 |
| 写入口径 | 写接口只管理语言目录、资源包和语言偏好，不写业务字段翻译值 |
| 幂等策略 | 语言建档、资源包发布、批量导入、偏好切换都必须支持业务键幂等 |
| 权限模型 | 至少区分平台级语言治理、租户级语言治理、个人偏好修改三类权限 |
| 审计留痕 | 发布、回滚、默认语言切换、回退链调整、批量导入导出必须进入统一审计 |
| 错误模型 | 统一错误码，区分语言不存在、语言未启用、回退链非法、资源键冲突、占位符错误、批次冲突 |
