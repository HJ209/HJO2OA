# I18n 前端说明

## 页面清单

| 页面/区域 | 页面职责 | 权限要求 | 状态要点 | 移动端约束 |
|-----------|----------|----------|----------|------------|
| 语言目录页 | 管理启用语言、默认语言、回退关系和作用域语言策略 | 平台管理员、租户管理员 | 已启用、已停用、默认中、回退冲突 | 移动端不提供目录治理 |
| 资源包列表页 | 查看资源包清单、模块归属、版本状态和最近变更 | I18n 管理员、审计员 | 草稿、已发布、已废弃、待校验 | 移动端仅建议只读查看 |
| 资源项编辑与对比页 | 编辑资源键值、查看版本差异、执行发布或回滚 | I18n 管理员 | 编辑中、校验失败、可发布、已回滚 | 移动端不提供批量编辑和对比 |
| 语言偏好页 | 个人切换语言，管理员代设租户或作用域默认语言 | 用户本人、平台管理员、租户管理员 | 跟随系统、个人覆盖、切换中、切换失败 | 移动端保留个人切换能力 |
| 校验与发布历史页 | 查看导入批次、缺失键、占位符错误和发布历史 | I18n 管理员、审计员 | 校验通过、校验失败、导入中、已回滚 | 移动端只看摘要，不支持导入发布 |

## 页面职责与关键交互

| 页面/区域 | 关键交互 | 说明 |
|-----------|----------|------|
| 语言目录页 | 启停语言、设置默认语言、配置回退链 | 回退链编辑时必须实时校验环路和默认语言可达性 |
| 资源包列表页 | 按模块/语言筛选、查看版本、跳转编辑 | 列表需突出模块、语言、版本、更新时间和状态 |
| 资源项编辑与对比页 | 单条编辑、批量导入、版本对比、发布、回滚 | 发布前必须完成占位符和重复键校验；回滚需展示影响范围 |
| 语言偏好页 | 切换个人语言、查看作用域默认值、恢复跟随系统 | 个人偏好切换应立即刷新界面上下文，但不修改资源包本身 |
| 校验与发布历史页 | 查看缺失键、导入错误、发布记录和审计摘要 | 历史只读展示，不允许从历史页直接修改资源内容 |

## 前端约束

| 项 | 说明 |
|----|------|
| 权限 | 必须区分平台级资源治理、租户级语言策略、个人偏好修改三类权限 |
| 状态 | 前端需清晰区分资源包状态与语言偏好状态，避免把“语言切换成功”误认为“资源包已发布” |
| 终端 | 一期移动端仅保留个人语言切换和资源查看，不支持批量导入、版本对比和发布回滚 |
| 治理 | 导入、发布、回滚、默认语言切换必须要求填写原因并进入统一审计 |
| 边界 | 前端只服务共享语义资源治理，不承担业务字段翻译工作台，该职责归 `HJO2OA-DataI18n` |
