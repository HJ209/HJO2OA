# 数据字典前端

## 页面清单

| 页面/区域 | 适用端 | 页面职责 |
|-----------|--------|----------|
| 字典类型管理页 | 管理端 | 展示字典类型列表、分类、层级开关、租户归属和最近更新时间。 |
| 字典项管理页 | 管理端 | 维护字典项树、排序、启停、多语言展示和批量操作。 |
| 导入导出弹窗 | 管理端 | 发起模板导入、导出任务并展示失败明细。 |
| 字典变更记录抽屉 | 管理端 | 查看最近变更摘要、操作者和审计关联号。 |
| 字典选择器/级联选择器组件 | 管理端、业务端 | 供业务表单、筛选器、门户组件只读引用字典项。 |

## 页面职责与关键交互

| 页面/区域 | 关键交互 | 约束 |
|-----------|----------|------|
| 字典类型管理页 | 新建、编辑、筛选、进入字典项详情 | 不允许在列表页直接删除已被引用的类型。 |
| 字典项管理页 | 新增子项、拖拽排序、启停、批量导入、多语言值编辑 | 层级拖拽后必须重新校验父子关系，非法调整不可提交。 |
| 导入导出弹窗 | 上传模板、预校验、查看失败项、重新提交 | 重复提交同一文件应命中幂等结果。 |
| 字典选择器组件 | 根据 `code` 拉取选项、按语言显示、按启用态过滤 | 组件只读引用，不暴露字典写入口。 |
| 变更记录抽屉 | 查看差异摘要、定位审计记录 | 只展示脱敏后的治理信息。 |

## 权限、状态与终端治理约束

- 权限约束：
  - 字典管理员可编辑类型和项。
  - 普通业务用户只可通过选择器组件读取。
  - 导入、批量排序、手工修复属于高风险治理动作，应要求更高权限。
- 状态约束：
  - 类型状态以 `ACTIVE / DISABLED` 展示，停用类型默认不再返回给新增业务表单。
  - 字典项状态以 `enabled=true/false` 展示，已停用项在历史详情中可见、在新增选择器中默认隐藏。
  - 层级字典在树模式下必须保证单根或明确父子归属，不展示成环数据。
- 终端约束：
  - 一期仅提供管理端治理页面。
  - 业务端和移动端只使用字典选择组件，不提供字典维护入口。
- 治理约束：
  - 前端不得将本地缓存视为真相源，收到刷新信号或版本变更后必须重新查询。
  - 所有批量写操作提交前应弹出影响范围确认，并显示将写入统一审计的提示。
