# Attachment 前端说明

## 页面清单

| 页面/区域 | 页面职责 | 权限要求 | 状态要点 | 移动端约束 |
|-----------|----------|----------|----------|------------|
| 附件资产列表页 | 查看附件元数据、绑定摘要、版本摘要和预览状态 | 业务操作人、附件管理员、审计员 | 上传中、预览中、可用、待删除、已删除 | 移动端保留列表与详情查看 |
| 上传与绑定面板 | 完成上传、绑定业务对象、查看上传进度和校验结果 | 业务操作人、附件管理员 | 待上传、上传中、校验失败、已建档 | 移动端支持单文件上传和基础绑定 |
| 预览与版本页 | 查看预览结果、版本链、下载记录和版本切换摘要 | 业务操作人、附件管理员 | 预览中、预览失败、当前版本、历史版本 | 移动端支持预览和历史版本查看，不支持复杂治理 |
| 配额与治理页 | 查看配额占用、预警、孤儿文件候选和删除任务 | 存储管理员、运维管理员、审计员 | 正常、预警、待清理、清理失败 | 移动端只看摘要，不支持治理操作 |
| 审计抽屉 | 查看上传、绑定、删除、版本创建和清理操作时间线 | 附件管理员、审计员 | 只读时间线、异常标记、治理备注 | 移动端只读展示 |

## 页面职责与关键交互

| 页面/区域 | 关键交互 | 说明 |
|-----------|----------|------|
| 附件资产列表页 | 按业务对象、文件类型、预览状态筛选，查看绑定和版本摘要 | 列表需突出“只持有元数据与绑定关系”，不要伪装成业务主档 |
| 上传与绑定面板 | 拖拽上传、分片上传、完成建档、选择绑定角色 | 上传完成后先建档再绑定，不允许前端直接写对象存储元数据 |
| 预览与版本页 | 预览文件、查看版本历史、创建新版本、下载授权 | 新版本创建要明确是追加式版本，不允许覆盖旧版本 |
| 配额与治理页 | 查看预警、确认清理、查看孤儿文件候选、补录治理备注 | 清理前必须展示活跃绑定校验结果和二次确认 |
| 审计抽屉 | 查看操作时间线、失败原因和治理记录 | 审计视图只读，不允许在时间线内直接修改元数据 |

## 前端约束

| 项 | 说明 |
|----|------|
| 权限 | 必须区分业务上传、附件治理、存储运维、审计查看四类权限 |
| 状态 | 前端需明确区分上传状态、预览状态、绑定状态和删除治理状态，避免混淆 |
| 终端 | 一期移动端只保留上传、预览、下载和基础绑定，不支持配额治理、孤儿文件清理和批量删除 |
| 治理 | 删除、物理清理、预警确认、预览重试都必须二次确认并写入审计 |
| 边界 | 前端只治理附件元数据和绑定关系，不建设业务生命周期中心，文件二进制仍由对象存储承载 |
