# Attachment 接口说明

## 接口分组

| 接口分组 | 关键查询 | 关键写入 | 分页与筛选 | 幂等要求 | 权限要求 | 审计要求 | 错误处理 |
|----------|----------|----------|------------|----------|----------|----------|----------|
| 上传与元数据建档 | 查询附件列表、详情、上传会话状态、对象存储回调结果 | 初始化上传、完成上传、建档元数据、逻辑删除申请 | 支持按 `filename`、`contentType`、`tenantId`、`businessType`、`previewStatus`、创建时间分页筛选 | 上传按 `clientUploadId` 或 `checksum + sizeBytes + tenantId` 幂等；完成建档按 `storageKey` 幂等 | 业务写入人、附件管理员、系统上传回调账号可写 | 初始化上传、完成上传、逻辑删除申请必须留痕 | `422` 文件类型不支持或校验摘要错误，`409` 重复建档，`404` 上传会话不存在 |
| 绑定关系与访问摘要 | 查询绑定列表、业务对象附件清单、访问模式、下载摘要 | 绑定附件、解绑附件、调整绑定角色、生成下载授权 | 支持按 `businessType`、`businessId`、`bindingRole`、`active`、`tenantId` 分页筛选 | 按 `attachmentId + businessType + businessId + bindingRole` 幂等写入 | 业务对象责任人、附件管理员可写；审计员可读 | 每次绑定、解绑、下载授权生成都必须记录来源和操作人 | `404` 业务引用不存在，`409` 绑定冲突，`403` 越权访问 |
| 版本与预览管理 | 查询版本历史、预览状态、预览失败原因、下载历史 | 创建新版本、重试预览、标记预览结果、生成预览票据 | 支持按 `attachmentId`、`versionNo`、`previewStatus`、时间区间筛选 | 新版本创建按 `attachmentId + sourceChecksum` 或请求号幂等 | 附件管理员、业务对象责任人、预览服务账号可写 | 新版本、预览重试、预览失败确认都必须审计 | `409` 版本冲突，`422` 预览类型不支持，`404` 附件或版本不存在 |
| 配额与删除治理 | 查询配额占用、预警列表、孤儿文件候选、删除任务状态 | 确认删除、确认物理清理、确认配额预警、补录治理备注 | 支持按 `tenantId`、`ownerType`、`warningLevel`、`cleanupStatus`、时间区间筛选 | 删除与清理按 `attachmentId + operationNo` 幂等；预警确认按 `warningId` 幂等 | 存储管理员、运维管理员可写；审计员只读 | 删除、清理、预警确认、备注必须进入统一审计 | `409` 仍有活跃绑定，`422` 配额规则缺失，`500` 存储清理失败需告警 |

## 接口治理约束

| 项 | 约束说明 |
|----|----------|
| 查询口径 | 列表接口统一返回附件元数据、绑定摘要、版本摘要、预览状态和最近治理动作 |
| 写入口径 | 写接口只管理元数据、版本、绑定和治理动作，不管理业务生命周期或业务权限真相源 |
| 幂等策略 | 上传完成、绑定、版本创建、删除治理都必须支持业务键幂等 |
| 权限模型 | 至少区分业务写入、附件治理、存储运维、审计查看四类权限 |
| 审计留痕 | 上传完成、绑定变更、版本创建、删除、物理清理、配额确认都必须进入统一审计 |
| 错误模型 | 统一错误码，区分上传校验失败、绑定冲突、预览失败、活跃绑定阻断、配额异常五类错误 |
