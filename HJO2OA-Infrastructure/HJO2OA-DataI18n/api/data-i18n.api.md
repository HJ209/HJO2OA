# DataI18n 接口说明

## 接口分组

| 接口分组 | 关键查询 | 关键写入 | 分页与筛选 | 幂等要求 | 权限要求 | 审计要求 | 错误处理 |
|----------|----------|----------|------------|----------|----------|----------|----------|
| 翻译项查询与 locale 解析 | 查询翻译项列表、详情、历史状态和按 locale 解析结果 | 批量解析请求、刷新解析缓存 | 支持按 `entityType`、`entityId`、`fieldName`、`locale`、`translationStatus`、`tenantId` 分页筛选 | 查询天然幂等；批量解析按请求摘要幂等 | 翻译员、审核人、源模块服务账号可查 | 批量解析和导出需记录调用来源 | `404` 翻译项不存在，`422` locale 非法或不支持 |
| 翻译项写入与审核 | 查询待审核条目、变更历史、原值对照 | 新增/更新译文、标记审核通过或驳回、重开翻译 | 支持按 `translationStatus`、`updatedBy`、`locale`、时间区间筛选 | 按 `entityType + entityId + fieldName + locale` 幂等写入；审核操作按 `translationId + reviewNo` 幂等 | 翻译员可编辑，审核人可审核，管理员可重开 | 译文修改、审核结论、重开动作必须全量审计 | `409` 状态冲突，`422` 译文格式非法，`404` 条目不存在 |
| 批量导入导出 | 查询导入批次、导出任务、校验结果和冲突摘要 | 批量导入、批量导出、执行覆盖确认、下载错误报告 | 支持按 `batchNo`、`entityType`、`locale`、`batchStatus`、时间区间筛选 | 导入按 `batchNo` 或 `fileChecksum` 幂等；覆盖确认按 `batchNo + confirmNo` 幂等 | 翻译管理员、审计员可用 | 导入、导出、覆盖确认、错误报告下载必须留痕 | `400` 文件格式非法，`409` 批次冲突，`422` 唯一键冲突或 locale 不支持 |
| 缺失项与完成度治理 | 查询缺失翻译、完成度统计、按对象聚合的语言覆盖率 | 标记完成、忽略缺失项、重建候选、补录治理备注 | 支持按 `entityType`、`entityId`、`locale`、`completionStatus`、`ownerModule` 筛选 | 完成标记按 `entityType + entityId + completionVersion` 幂等 | 翻译管理员、审核人可写 | 完成确认、忽略缺失、重建候选都必须写审计 | `404` 对象不存在，`422` 源字段未注册，`409` 已完成对象重复确认 |

## 接口治理约束

| 项 | 约束说明 |
|----|----------|
| 查询口径 | 列表接口统一返回原值、当前译文、回退结果、审核状态和最近更新时间，避免前端再拼字段 |
| 写入口径 | 写接口只维护翻译值和治理状态，不直接修改业务对象原值 |
| 幂等策略 | 翻译写入、批量导入、完成确认、候选重建都必须支持业务键幂等 |
| 权限模型 | 至少区分翻译编辑、翻译审核、批量治理、只读查询四类权限 |
| 审计留痕 | 译文修改、审核结论、覆盖确认、完成确认和缺失项忽略必须进入统一审计 |
| 错误模型 | 统一错误码，区分翻译项不存在、唯一键冲突、locale 非法、状态冲突、批次冲突五类错误 |
