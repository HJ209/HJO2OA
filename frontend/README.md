# frontend

`frontend/` 是项目的标准 React 前端工程骨架目录，用于承载实际可运行的 Web、门户与移动端前端代码。

## 当前骨架定位

- `apps/`
  - 用于承载门户端、管理端、移动端等 React 应用入口
  - 当前已提供 `portal-web/` 作为首个可运行的 React + Vite 应用
- `packages/`
  - 用于承载共享组件、SDK、工具函数、表单渲染协议实现等复用包
- `public/`
  - 用于承载静态资源

## 约束

- 前端目录结构与模块边界应与根目录业务模块中的设计文档保持一致
- 前端统一采用 React 技术栈，不再引入 Vue 相关框架依赖
- 具体前端工程拆分方式以后续前端架构方案为准
