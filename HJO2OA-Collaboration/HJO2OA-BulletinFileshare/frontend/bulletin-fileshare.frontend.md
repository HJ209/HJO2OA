# bulletin-fileshare 前端说明

## 页面清单

| 页面 | 主要角色 | 页面职责 | 移动端口径 |
|------|----------|----------|------------|
| 公告场景页 | 全员、授权人员 | 浏览公告、制度、知识等已发布内容入口。 | 重点支持浏览与跳转阅读。 |
| 共享空间列表页 | 空间成员、组织管理员 | 查看可访问的共享空间及资源概况。 | 提供轻量列表。 |
| 共享空间详情页 | 空间成员、管理员 | 查看空间成员、资源绑定、入口分类。 | 支持资源浏览与打开。 |
| 成员权限抽屉 | 空间管理员 | 维护成员主体和权限级别。 | 移动端不开放编辑。 |
| 资源绑定抽屉 | 空间管理员、编辑者 | 绑定内容、附件、链接并调整排序。 | 移动端仅浏览，不开放复杂绑定。 |

## 页面职责与关键交互

| 页面 | 职责 | 关键交互 | 权限与状态约束 |
|------|------|----------|----------------|
| 公告场景页 | 汇总展示可见范围内的已发布内容入口。 | 按内容类型、发布时间、关键字筛选，点击跳转 `03` 正文页。 | 不提供正文编辑；内容可见性完全复用 `03/01`。 |
| 共享空间列表页 | 展示用户可进入的共享空间。 | 查看空间状态、资源数量、进入详情。 | `CLOSED` 空间只读显示。 |
| 共享空间详情页 | 呈现空间成员与资源绑定结果。 | 浏览资源分组、打开附件预览、跳转内容正文、复制外部链接。 | `READ_ONLY` 空间禁用编辑入口。 |
| 成员权限抽屉 | 管理共享范围与角色。 | 新增/删除成员主体、调整 `VIEWER/EDITOR/MANAGER` 权限。 | 仅空间管理员可用，且 `CLOSED` 空间不可编辑。 |
| 资源绑定抽屉 | 管理场景入口资源。 | 绑定内容、附件、链接，拖拽排序，清理失效绑定。 | 仅管理员/编辑者可操作；失效引用必须明确标识。 |

## 权限、状态、移动端约束

| 维度 | 约束 |
|------|------|
| 权限 | 统一复用 `01` 主体权限和空间成员权限；前端不缓存脱离空间权限的资源列表。 |
| 状态 | 空间状态仅支持 `ACTIVE/READ_ONLY/CLOSED` 三类，页面交互必须与状态一致。 |
| 内容边界 | 公告正文编辑、发布和检索统一跳转或复用 `03`，本模块页面只做场景装配。 |
| 移动端 | 覆盖公告浏览、空间进入、资源打开；成员维护和复杂绑定编辑以 PC 为主。 |
