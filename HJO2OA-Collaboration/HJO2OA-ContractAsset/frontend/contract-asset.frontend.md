# contract-asset 前端说明

## 页面清单

| 页面 | 主要角色 | 页面职责 | 移动端口径 |
|------|----------|----------|------------|
| 合同台账页 | 合同经办人、组织管理员 | 查看合同列表、状态、到期时间和履约概况。 | 提供轻量列表与到期提醒。 |
| 合同详情页 | 合同经办人、审计角色 | 查看合同主档、履约节点、提醒规则和附件摘要。 | 支持详情阅读。 |
| 履约与提醒页 | 合同经办人、组织管理员 | 维护履约节点、查看临期提醒和处理进度。 | 移动端只读查看。 |
| 资产台账页 | 资产管理员、保管人 | 查看资产列表、状态、保管人和分类。 | 提供我的资产视图。 |
| 资产流转页 | 资产管理员、保管人 | 发起流转、查看流转历史和处理状态。 | 支持领用/归还等轻量操作。 |

## 页面职责与关键交互

| 页面 | 职责 | 关键交互 | 权限与状态约束 |
|------|------|----------|----------------|
| 合同台账页 | 聚合合同全量台账与筛选入口。 | 按 `contractNo/status/effectiveTo/ownerOrgId` 筛选，进入详情与提醒页。 | 仅授权组织范围可见；归档合同默认只读。 |
| 合同详情页 | 展示合同主档与履约上下文。 | 查看主档字段、履约节点、附件摘要、审批摘要。 | `DRAFT/ACTIVE/EXPIRING` 可按权限编辑，`ARCHIVED` 只读。 |
| 履约与提醒页 | 管理到期与履约节点闭环。 | 维护节点状态、查看提醒命中记录、进入处理详情。 | 仅合同经办人与管理员可编辑。 |
| 资产台账页 | 聚合资产状态和责任归属。 | 按 `assetCode/status/category/custodianId` 筛选，进入详情与流转。 | 资产管理员可看全量，保管人仅看本人范围。 |
| 资产流转页 | 承接资产状态变化动作。 | 发起领用/借用/归还/报废，查看流转时间轴。 | 已报废资产不可再领用或借用；批量盘点以 PC 为主。 |

## 权限、状态、移动端约束

| 维度 | 约束 |
|------|------|
| 权限 | 统一复用 `01` 的组织、岗位和人员权限；金额等敏感字段按角色脱敏展示。 |
| 状态 | 合同和资产的按钮可用性必须以业务主档状态为准，不能依赖前端本地推断。 |
| 审批 | 审批动作统一跳转 `02`；本模块页面只展示审批摘要和回写结果。 |
| 移动端 | 覆盖台账查询、到期提醒、轻量流转；复杂节点配置、批量盘点和导入导出以 PC 为主。 |
